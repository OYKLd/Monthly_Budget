# Monthly_Budget

## Description
Application web permettant de gérer un budget mensuel en enregistrant les revenus et les dépenses par catégorie, avec calcul automatique du solde restant.  
Ce projet est le **vingt-cinquième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Structurer des données financières
- Calculer des totaux dynamiques (revenus, dépenses, solde)
- Gérer des catégories de dépenses
- Concevoir un mini dashboard financier
- Manipuler des tableaux d’objets complexes

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage
- (Optionnel) Chart.js

---

## Fonctionnalités
- Définition d’un revenu mensuel
- Ajout d’une dépense (titre, montant, catégorie)
- Calcul automatique :
  - Total des dépenses
  - Solde restant
- Affichage des dépenses sous forme de liste
- Suppression d’une dépense
- Sauvegarde locale des données

---

## Design & UX
- Dashboard financier en haut de page (revenu, dépenses, solde)
- Cartes statistiques bien mises en évidence
- Liste structurée et lisible
- Code couleur simple (positif / négatif)
- Responsive (mobile et desktop)
